учебная программа go-dictionary

Запуск осуществляется через main.go
Имеет предопределенный словарь из 3х слов
Команды обрабатываются в терминальном режиме. Команды не требовательны к регистру

Структура проекта содержит 3 пакета. 

Main - основной
Dictionary - набор функций и методов, отвечающих за логику программы
Input - набор функций ввода пользовательских данных
